## To run locally, add a `.env.local` to your root directory with the following enviornment variables set:

* `REACT_APP_MAPS_API_KEY`
* `REACT_APP_GOOGLE_MAP_KEY`
* `REACT_APP_PLAYFAB_GAME_ID`
